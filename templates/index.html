Social Connects

YouTube Not connected

Sign in with Google to publish Shorts directly, auto-sync views & likes, and learn from what goes viral.

⚙ Server setup required — add Google OAuth credentials (see SOCIAL_SETUP.md) to enable real sign-in.

Instagram Not connected

Connect an Instagram Business account to publish Reels, auto-sync plays & likes, and optimize your strategy.

⚙ Server setup required — add Meta app credentials (see SOCIAL_SETUP.md) and link an IG Business account.

Facebook Not connected

Connect a Facebook Page to publish videos, monitor performance, and learn from top-performing content.

⚙ Server setup required — add Meta app credentials (see SOCIAL_SETUP.md) to enable Page publishing.

◉ How publishing & analytics work
  • Click Connect to sign in via the platform's official OAuth — tokens are stored securely on the server.
  • Schedule reels from the Calendar; the server publishes them automatically at the set time.
  • Published posts auto-appear in Insights; hit Sync analytics to pull live views/likes.
  • The AI then surfaces what's working in Compare.
  • One-time setup of API credentials is required — see SOCIAL_SETUP.md.
